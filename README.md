# wprobe

Что это вообще?
===============

Набор программ для демонстрации и отладки работы с ВАХ ящиком.
Подключение к BAX ящику по Modbus.

scan.m - скрипт для выполнения сканирования напряжения смещения. Умеет:
            - сканировать, изменяя уставку ЦАП
            - считывать измеренное напряжение смещения по АЦП
            - считывать измеренный ток по АЦП
            - считывать для сравнения при необходимости напряжение по 
                внешнему эталонному вольтметру или ток по внешнему амперметру.
                подключение подразумевается через TCP сокет, протокол VISA
            - автоматически выбирать предел измерения по току
            - сохранять результаты и конфигурацию каждого скана в отдельном 
                каталоге в out/

plotiscan.m - скрипт для вывода результатов сканирования с подключенным
                эталонным амперметром

plotvscan.m - скрипт для вывода результатов сканирования с подключенным
                эталонным вольтметром

plotchar.m - скрипт для вывода ВАХ

Порядок работы
==============

1. Выполнить сканирование. Для этого сделать все нужные настройки, 
    изменяя значения переменных в скрипте scan.m (в начале).
    Затем запустить скрипт и дождаться пока он закончит работу.
    В конце скрипт попросит ввести краткое описание скана (многострочный ввод,
    чтобы закончить ввод - ввести пустую строку).
    Результат и соответствующая конфигурация будут записана в out/ в каталог вида
    scan_vv_NNNN_YYYY-MM-DD, где NNNN - номер запуска. Номера автоматически
    назначаются по возрастанию. Описание будет сохранено в readme.txt там же.

2. Просмотреть результаты сканирования.
    Для этого запустить один из перечисленных выше скриптов. Перед запуском
    задать в начале скрипта нужный номер набора результатов, поправив значение
    переменной scanNumber в начале.


Все написано и работало в Matlab R2023b Update 3.
